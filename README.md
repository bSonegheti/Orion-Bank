💳 Banco Digital - Projeto de Estudo
Este projeto é um Banco Digital desenvolvido puramente com HTML5, CSS3 e JavaScript, com lógica básica funcional. O objetivo principal é demonstrar conhecimentos em desenvolvimento front-end e organização de lógica para simular operações simples de um sistema financeiro.

🎯 Objetivo
Projeto criado exclusivamente para portfólio e currículo, com foco em aprendizado e evolução contínua. Novas funcionalidades e melhorias serão adicionadas com o tempo.

🚀 Tecnologias Utilizadas
HTML5
CSS3
JavaScript (Vanilla)

⚙️ Funcionalidades
Front End de banco digital
Lógica básica de operações
Estrutura inicial funcional
Projeto preparado para melhorias futuras

📌 Observação Importante
Este projeto não é um banco real e não deve ser utilizado para fins financeiros reais.
Trata-se apenas de uma simulação com fins educacionais e demonstração de habilidades.

📈 Status do Projeto
🛠️ Em desenvolvimento — melhorias serão adicionadas continuamente.
